# Purshoottam Mishra Portfolio

Professional MERN portfolio for Purshoottam Mishra, built with React, Vite, Tailwind CSS, Node.js, Express, and MongoDB.

## Run Backend

```bash
cd server
npm install
npm run dev
```

## Run Frontend

```bash
cd client
npm install
npm run dev
```

The frontend runs on `http://localhost:5173` and proxies `/api` requests to the backend on `http://localhost:5000`.
